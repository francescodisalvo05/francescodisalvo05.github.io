Books for Data Science and Machine Learning

Hey friends, I am currently writing this post to pass the time needed for training my model, and if you are a data scientist (or something similar), you can perfectly understand how it feels.

Today I am going to talk about three interesting books about Data Science and Machine Learning. There are many amazing resources online but, personally, when I start learning something new, I always prefer to start from a good book. I want to report a quote from Cassie Kozyrkov (Lead of Decision Intelligence at Google) that I recently saw at an amazing ML conference:

"The world represented by your data is the only world you can expect to succeed in"

She wanted to say that you will learn exactly what you find in your resources (textbooks or whatever), and as a consequence, it is fundamental to choose the best ones. I am not a Senior Data Scientist or a Senior Machine Learning Engineer, but I can truly recommend these three books:


  • Python for Data Analysis, Wes McKinney

  • Hands-On Machine Learning with Scikit-Learn,Keras and Tensorflow, Aurélien Géron

  • Approach Almost any Machine Learning Problem, Abhishek Thakur

In my opinion these are some great resources, I don’t know if they are the best ones at all, but I can say that I learnt a lot from them. I studied from "Python for Data Analysis" around one or two years ago, when I started being interested in the amazing world of Data Science. By looking at the contents and the reviews it seemed a valid option and I can confirm that. It covers: Python data structures, Numpy, Pandas, Data cleaning, Time series, Matplotlib. It is really clear and there are several use cases for learning some best practises.

The second one is another great book but it is really technical, there is a lot of "math" and each model is accurately explained. So, if you have skills, or at least you like math and you’re not afraid to lose some extra time, this is a great option for Machine Learning. It covers most of the existing models for supervised and unsupervised problems with clean and detailed explanations. Then, the second part is devoted to Neural Network and Deep Learning. I haven't studied this part yet, I just read some chapters just for curiosity and it seems really cool as the first one. Maybe I will update the post once I will complete it.

The third one is a “best-practises” book by a Kaggle Grandmaster. Here there is a huge debate about Kaggle and the real word, in fact the kaggle competitions take care about the model scores and nothing else, no matter how complex the model is. It is definitely true, but actually I found some best practises that I didn’t know, so I can say that it is definitely worth it!

I hope you found this post useful, see you next time,
